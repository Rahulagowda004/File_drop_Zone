# **App Name**: File Drop Zone

## Core Features:

- File Upload: Users can upload a file associated with a unique keyword.  Backend is implemented using FastAPI.
- Public File Access: The uploaded file is publicly accessible at https://www.project.com/{keyword}.
- Homepage Search: Homepage at https://www.project.com features a single search bar for keyword lookup.
- Automatic File Deletion: After 24 hours, the file is automatically deleted from both storage and the database.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6), reminiscent of clear skies and digital spaces, to convey trustworthiness and ease of use.
- Background color: A light, almost white blue (#E3F2FD) to provide a clean and unobtrusive backdrop.
- Accent color: A muted green (#81C784) used sparingly for highlights and calls to action, suggesting accessibility and confirmation.
- Clean, sans-serif fonts to ensure readability and a modern feel. Focus on fonts optimized for web use to maintain performance.
- Minimalist layout featuring a central search bar on the homepage and clean file display pages.
